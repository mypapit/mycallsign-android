Mycallsign-android

Handy app for looking up Malaysian Hamradio callsign

View on GitHub Download .zip Download .tar.gz

Introduction

MYCallsign is an Android application for looking up Malaysian Amateur Radio (hamradio) callsign. User only need to key-in the ham callsign that they wished to view and the application will automatically display relevant information related to the hamradio operator.

Advatages:

  • Database is updated from time to time according to SKMM website
  • Does not require internet connection
  • Supports quick lookup of operator's callsign and handle

Supported Platform

Currently MYCallsign-android is supported on Android 4.1 (Jelly Bean) and above.

Using Android Ice Cream Sandwich, Honeycomb, Gingerbread and Froyo? Then check out MYCallsign Lite
